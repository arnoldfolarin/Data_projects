# Turning Event Logs into Business Metrics

### **Overview**
Congratulations, you’ve been hired as a junior analyst at an e-commerce company! Your task is to analyze raw transaction logs from the company’s website and turn them into actionable business metrics. The dataset captures user activities such as viewing product pages, opening shopping carts, and completing purchases.

### **Contents**
- **Retention_Rates_Over_Time.xlsx**: An Excel file containing:
  - Processed retention rate data for purchases.
  - Metrics derived from raw user activity logs.

### **Key Objectives**
1. Explore the “raw_user_activity” data.
2. Extract meaningful metrics from the event logs.
3. Present retention rates of purchases over time.

### **Tools Used**
- **Excel**: For data analysis and calculating retention rates.

### **Usage**
1. Download the **Retention_Rates_Over_Time.xlsx** file.
2. Open it with any spreadsheet software (e.g., Microsoft Excel, Google Sheets).
3. Review the retention rates and derived business metrics.
